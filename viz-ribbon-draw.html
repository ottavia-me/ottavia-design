Each ribbon is an SVG <path>. We set stroke-dasharray to the path length and animate stroke-dashoffset from full length to 0. The dash travels along the path in its authored direction, so even curves that move backward or cross each other appear as if drawn by hand. Same technique works in React Native via react-native-svg + reanimated.
