The chat block palette

Reverse exercise: instead of designing content and finding components, define the vocabulary of blocks Ottavia composes from, so she has enough to draw the best answer for any wellness intent. Six real responses (Elad UAT data), then the 12-block palette, then the ledger. Built on the v1 Visual Refresh tokens, Modern Era, green-anchored.
